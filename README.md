# Ayush Pandey | Portfolio

Welcome to my personal portfolio website. This is a minimal, premium, dark-themed portfolio built to showcase my developer profile, skills, projects, and experiences.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Smooth Scroll:** Lenis

## Getting Started
To run the development server locally:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
